General

Estructura

  • La pagina comenzara con <!Doctype HTML> para definir el tipo de archivo y despues de este le seguira <html></html> que sera la raiz de la pagina y contendra la estructura de la pagina

  • <head></head> es la cabecera de la pagina en donde se colocara el tag <title></title> que sirve para darle nombre a la pestaña de la pagina y se utilizara para referenciar diferentes archivos como, por ejemplo CSS o JavaScript

  • <body></body> es el cuerpo de la pagina y es la parte visible de la misma.

  • <h'numero'> es la etiqueta para las cabeceras. Hay desde la cabecera h1 hasta h6

  • Esto es un parrafo<p> </p>. Aqui se escribiran la mayor parte de las cosas. La etiqueta <pre></pre> es una etiqueta parecida a los parrafos pero respetando el formato del texto introducido (como saltos de lineas, etc...)

    Esto es un "pre" de esos

    Tambien en los parrafos se puede definir title="" para modificar el texto que se muestra cuando pasas el raton por encima

  • <br> sirve para insertar un salto de linea y <hr> sirve para insertar una linea horizontal en la pagina.

  • <button></button> Sirve para colocar un boton en html. Se podria usar para incrustar un vinculo

Hipervinculos <a>

Esto es un link

Los links <a></a> sirven para redirigirte a lugares concretos. Tienen varios atributos, los dos mas importantes son:

  • href="" -- En este atributo se especifica el link/ruta a la que se quiere acceder.
  • target="" -- En este atributo se especifica como/donde se va a abrir. El valor por defecto es en la misma ventana (_self) y _blank es en una nueva ventana.

Tambien se puede contener una imagen dentro de un link para que la imagen actue de medio o referenciar una ID de CSS para ir directamente a esa marca (bookmark)

Imágenes <img>

Esta etiqueta tiene varios atributos como src="" donde se indica la ruta de la imagen; el atributo alt="" donde se indica el texto que se mostrara si no hay imagen. Tambien estan los atributos width="" y height="" para controlar anchura y altura o utilizar el CSS

Se puede utilizar el atributo usemap="nombreDeMap" para crear una imagen con diferentes links en distintas coordenadas de la misma

"Mapeado Imagenes"<map>

Esta etiqueta complementa <img> y sirve para marcar coordenadas que referenciaran a distintos links.

  • Primero se abre la etiqueta <map>, se le añade el atributo name="" y como si fuese una tabla, dentro de esta se insertaran las etiquetas <area>
  • Aqui se definiran las diferentes formas con atributos como:
    shape = "rect | circle | poly | default"
    coords = "x, y, x2, y2
    alt = "textoAlternativo"
    href="link"

Elementos en bloque y en linea

Los elementos tiene una forma predeterminada de mostrarse, en bloque o en linea. Hay etiquetas que se usan para agrupar otras etiquetas y mostrarlas de la manera deseada como <div> o <span>.
A estas etiquetas se les puede aplicar ciertos estilos CSS para mostrar el contenido. Esta pagina es un claro ejemplo de los bloques <div>.

Iframes<iframe>

Los Iframes sirven para mostrar una pagina dentro de otra pagina, mediante una ULR. El atributo src="" sirve para especificar la pagina que cargara.

Layouts

Las paginas suelen contener varios elementos/menus. HTML ofrece elementos semanticos para catalogar las diferentes partes de la web.

  • <header>: Especifica el encabezado del documento.
  • <nav>: Especifica el contenedor con los links.
  • <article>: Especifica una parte independiente de un articulo.
  • <section>: Especifica las diferentes secciones del documento.
  • <aside>: Define contenido aparte del contenido.
  • <footer>: Define el pie de la seccion/documento.

Formularios

Se utiliza la etiqueta < form > para definir elementos que recogeran la entrada del usuario. Esto puede contener varios elementos como:

  • <input>: Dependiendo del atributo type puede ser de varias formas:
    • "text": Define una linea de texto como campo para la introduccion de datos. Hay que especificar un nombre al campo con el atributo "name".
    • "radio": Define un boton de seleccion tipo radio. Se le tiene que dar nombre y establecer su valor con el atributo "value".
    • "submit":Define un boton de seleccion de tipo submit. Se le tiene que dar tanto nombre como valor
