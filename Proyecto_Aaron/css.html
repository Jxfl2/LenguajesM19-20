General

Inroducción

CSS (Cascading Style Sheets) sirve para definir el estilo de la pagina. Se puede incluir en el tag, con el atributo style, en el mismo archivo, incluyendo el tag <style> en el head de la pagina, o en un archivo a parte, vinculando con <link rel="" href="" >

En el css se puede utilizar clases (referenciados por un ".") o id unicos ( referenciados por "#") para dar estilos propios u otras funcionalidades (cabeceras).

Cada etiqueta de html puede tener atributos propios, pero todas tienen el atributo style="" para poder establecer estilos CSS.

Hipervinculos

Los links <a> tienen "estados" que pueden ser editados con CSS. Como por ejemplo:

  • a:link -- Editar el link en el estado "base".
  • a:visited -- Edita el link en el estado "visitado".
  • a:active -- Edita el link en el estado "activo" (cuando se esta haciendo click en él).
  • a:hover -- Edita el link en el estado "sobre el" (cuando pasas el raton por encima).

Listas

Aqui hay algun atributo util para el CSS de las listas.

  • list-style-type: Define el estilo del "marcador" de la lista. Valores (disc | circle | square | none).
  • type : Define la enumeracion de las listas ordenadas. Valores ("1" | "A" | "a" | "I" | "i").
  • overflow: Para tratar los elementos que salen de la pagina. Valores hidden.
  • float: Para que los elementos pasen a una "capa" superior en la posicion idicada. Si el elemento siguiente se hace float tambien este se "pegua" al elemento anterior. Valores (left | right | none | inherit).
  • display: Sirve para mostrar los elementos en "linea" en vez de en forma de "bloque"

Float & clear

La propiedad float="" sirve para que los elementos pasen a una "capa" superior en la posicion idicada (no ocupando espacio pero si). Si el elemento siguiente se hace float tambien este se "pegua" al elemento anterior.

Valores:

  • left: Sirve para que el elemento se alinee arriba a la izquierda del elemento contenedor.
  • right: Sirve para que el elemento se alinee arriba a la derecha del elemento contenedor.
  • none: Sirve para que el elemento no "flote".
  • inherit: Sirve para que el elemento adquiera el valor de su "padre".

La propiedad clear="" sirve para especificar los elementos flotantes que no se permiten a los lados.

Valores:

  • none: Permite elementos flotantes a ambos lados (por defecto).
  • left: No permiten elementos flotantes a su izquierda.
  • right: No permiten elementos flotantes a su derecha.
  • both: No se permiten elementos flotantes a ambos lados.
  • inherit: Adquiere el valor de su "padre".

Display

La propiedad display sirve para mostrar un elemento de una manera en concreto (o no mostrarlo...).

Todos los elementos poseen valores por defecto de esta propiedad, algunos tipo "block" como el <p> y otros tienen el valor "inline" como <img>, incluso algunos no se muestran (none) como la etiqueta <script>.

Valores:

  • block: Muestra el elemento como un tipo "bloque".
  • inline: Muestra el elemnto como un tipo "en linea".
  • inline-block: Muestra el elemento como un tipo "en linea" pero pudiendo establecer altura y anchura.
  • none: No muestra el elemento y tampoco ocupa lugar en la pagina.

Atributos

El atributo style tiene varios valores como el background-color:... para cambiar el color del fondo de la etiqueta.

Incluso cambiar el tipo de fuente del texto con font-family:...; Es recomendable poner varias fuentes para que cualquier navegador tenga mas opciones para cargar.

Tambien se puede cambiar el color del texto con color:...;

Se puede cambiar el tamaño con font-size:...;

Y alinear el texto con text-align:left | center | right;

En HTML hay 140 nombres para colores ya establecidos como DodgerBlue. Si se desea poner cualquier otro color siempre se puede usara el sistema 'hex' , 'rgb' o 'hsl'.

El padding sirve para definir el espacio entre el elemento y su borde (relleno)

El margin sirve para definir el espacio fuera del borde del elemento.

Con los atributos width:"" y height:"" se puede controlar el tamaño de un elemento
